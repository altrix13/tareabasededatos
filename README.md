# tareabasededatos # Conexión de C++ con MySQL usando ODBC

Este proyecto demuestra cómo establecer una conexión entre un programa en C++ y una base de datos MySQL utilizando ODBC.

## Tecnologías utilizadas

- C++
- MySQL
- ODBC
- Visual Studio

## Descripción

El programa crea una conexión a una base de datos MySQL mediante un DSN configurado en el sistema operativo. Utiliza funciones de la API ODBC para establecer la conexión y verificar su funcionamiento.

## Funcionamiento

1. Se configura un DSN en ODBC
2. Se utiliza el DSN en el código C++
3. Se ejecuta el programa
4. Se muestra un mensaje de conexión exitosa
